# 🎵 Mood Music Suggester

Mood Music Suggester is an Android application that recommends songs based on the user’s mood.  
The app allows users to choose moods such as **Happy**, **Sad**, or **Relaxed**, and then displays a list of songs related to that mood.

The project is developed using **Java, XML, and Groovy** in **Android Studio** and demonstrates core Android concepts such as **RecyclerView, Adapters, and user interaction using Toast messages**.

---

## 📱 Features

- Select mood (Happy, Sad, Relaxed)
- View mood-based song recommendations
- Scrollable song list using RecyclerView
- Simple and user-friendly interface
- Toast message when a song is selected

---

## 🛠️ Technologies Used

- **Java** – Application logic
- **XML** – UI design
- **Groovy / Gradle** – Build configuration
- **Android Studio** – Development environment
- **RecyclerView** – Display song lists

## ⚙️ How It Works

1. The user opens the application.
2. The app displays mood options such as **Happy, Sad, and Relaxed**.
3. The user selects a mood.
4. The application shows a list of songs related to that mood using **RecyclerView**.
5. When a song is selected, the app displays a **Toast message** saying  
   **“Playing Song Name.”**
